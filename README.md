# Hamidreza Mafi — Personal Website

A dependency-free personal website with a reversible, scroll-driven character physics scene,
interactive project-card colliders, and a focused contact page.

Live at [thehamidrezamafi.github.io](https://thehamidrezamafi.github.io/).

## Run locally

```bash
npm run dev
```

Then open [http://localhost:4173](http://localhost:4173).

## Personalize

- Terminal copy lives in `TERMINAL_LINES` near the top of `app.js`.
- Project cards and contact details live in `index.html`.
- Colors, typography, and spacing tokens are at the top of `styles.css`.

No install or build step is required. The included server is only for local preview; GitHub Pages serves the website directly from the root of `main`.
